# BCG Virtual Internship

## Overview
This project is based on the BCG Data Science Virtual Internship. It involves exploratory data analysis (EDA) and customer churn analysis for a client in the energy sector.

## Files
- `Data Description.pdf` — Description of the dataset and its features
- `Task 2 - eda_starter.ipynb` — Jupyter notebook for exploratory data analysis
- `client_data.csv` — Client information dataset
- `price_data.csv` — Historical price data dataset

## Goals
- Perform exploratory data analysis on client and price data
- Identify patterns and insights relevant to customer churn

## Procedure
1. Load and inspect the client and price datasets
2. Perform data cleaning and feature engineering
3. Conduct exploratory data analysis (EDA) with visualizations
4. Summarize findings and insights

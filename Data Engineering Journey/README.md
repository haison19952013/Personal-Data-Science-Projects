# Data Engineering Journey

This featured project highlights a separate portfolio repository focused on hands-on data engineering practice, covering batch processing, streaming systems, analytics engineering, and cloud-oriented workflows.

- **Repository:** [haison19952013/Data-Engineering-Journey](https://github.com/haison19952013/Data-Engineering-Journey)
- **Focus areas:** Python, SQL, Bash, Kafka, Spark, PostgreSQL, MongoDB, Airflow, and dbt
- **Project types:** data wrangling, ETL, streaming pipelines, analytics engineering, and cloud data workflows

## Why it stands out

- Shows a progression from foundational exercises to end-to-end pipeline projects
- Combines data ingestion, transformation, orchestration, and serving patterns
- Includes practical portfolio pieces such as dbt analytics work, Kafka streaming pipelines, and GCP-based data workflows

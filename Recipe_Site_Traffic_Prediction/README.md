# Recipe Site Traffic Prediction

## Overview
This project focuses on predicting recipe site traffic based on recipe features. The goal is to identify which recipes are likely to generate high traffic so that the business can make data-driven decisions about which recipes to feature on the homepage.

## Files
- `Recipe Site Traffic Prediction.pdf` — Full project report
- `Recipe Site Traffic Prediction_presentation.pdf` — Presentation slides summarizing the project

## Goals
- Build a predictive model to classify whether a recipe will generate high traffic
- Provide actionable insights to drive business decisions on recipe promotion

## Procedure
1. Explore and clean the recipe dataset
2. Perform exploratory data analysis (EDA)
3. Build and evaluate classification models
4. Present findings and recommendations
